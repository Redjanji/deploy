XSS 微服务 API 测试平台
从左侧选择需要测试的 API 接口，填写参数后发送请求。
快速开始
1. 点击顶部「获取 App Token」按钮（HMAC-SHA256 自动签名）
2. 点击「快速登录」按钮（自动注册随机用户并获取 User Token）
3. 从左侧选择接口，填写参数后点击「发送请求」
认证说明
• App Token: 服务间调用认证（HMAC 签名，5分钟有效期）
• User Token: 用户 JWT 认证（登录后获取，Bearer 头）
• 需要用户权限的接口会自动携带 User Token
响应格式
统一格式: { "code": 200, "message": "success", "data": ... }
分页格式: { "records": [], "total": 0, "size": 20, "current": 1 }
注意事项
• 本页面需通过 HTTP 服务器访问（非 file:// 协议）
• 服务端时间戳验证有 ±5 分钟容差，本地时钟需准确
• 文件上传接口使用 multipart/form-data
